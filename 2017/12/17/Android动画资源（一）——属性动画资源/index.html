Android动画资源（一）——属性动画资源

2017.12.17 SuperRed  热度 ℃

Android动画资源系列文章：Android动画资源文件总结

Property Animation

  • 在XML中定义的一种动画，它修改目标对象的属性，例如背景颜色或透明度值。
  • 文件位置
    res/animator/filename.xml
    该文件名将作为资源ID。
  • 资源引用方式
    Java中：R.animator.filename
    XML中： @[package:]animator/filename
  • 语法
  • 注意
    • 属性动画资源文件只能有一个根标签，而且只能是，< objectanimator >或< ValueAnimator >中的一
      个。
    • 可以将多个动画标签组合起来，同时标签内也可以嵌套标签。

####

  • 一个能够包裹其它元素（, 或者其它 ）的容器。相当于Java代码中的AnimatorSet。
  • 每一个 都可以通过ordering属性定义自己动画的执行顺序。

  • 属性
    android:ordering 指定动画的播放顺序

    | 属性值 | 描述 |
    | ————- |————-|
    | sequentially | 动画顺序执行 |
    | together (默认值) | 动画一起执行 |

  • 在一段特定的时间内改变对象的特定属性。相当于Java代码中的ObjectAnimator。

  • 属性

    • android:propertyName
      属性值类型：String。
      属性名，例如：可以是alpha或者backgroundColor等等。
    • android:valueTo
      属性值类型：float，int，或者color。
      动画结束的属性值，可以使float，int或者color，如果是color类型，应该由六位十六进制数(例如, #333333)表示。
    • android:valueFrom
      属性值类型： float，int，或者color。
      动画开始的属性值
    • android:duration
      属性值类型：int。
      动画持续的时间，单位毫秒，默认值为300毫秒。
    • android:startOffset
      属性值类型：int。
      调用 start()方法以后延时的毫秒数。
    • android:repeatCount
      属性值类型：int。
      动画重复执行的次数。repeatCount 可以设置为正整数或者-1。repeatCount 设置为-1代表动画无 限循环；repeatCount 设置为一个正整数，例如1，代表在动画运行完成以后再重复运行一次，也就是总共运行两次。repeatCount 默认值是0，代表不重复执行动画。
    • android:repeatMode
      属性值类型：int。
      动画在动画结束时的表现方式。repeatcount必须设置为正整数或“-1”repeatMode这个属性才有效。repeatMode设置为“reverse”为倒序重复执行动画，repeatMode设置为“repeat”为正序重复执行动画。

    • android:valueType
      动画值类型。如果值为颜色，则不指定此属性，动画框架自动处理颜色值。valueType属性有intType和floatType 两种属性值。

      | 属性值 | 描述 |
      | ————- |————-|
      | intType| 指定动画值为int类型 |
      | floatType (默认值) | 指定动画值为float类型 |

  • 在指定的时间段执行动画。相当于Java代码中的ValueAnimator。

  • 属性

    • android:valueTo
      属性值类型：float,int或者color。
      动画结束的属性值，如果是color类型，应该由六位十六进制数(例如：#333333)表示。
    • android:valueFrom

    • android:startOffset

    • android:repeatCount

    • android:repeatMode

      | ————- |————-|

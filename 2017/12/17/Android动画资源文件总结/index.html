Android动画资源文件总结

2017.12.17 SuperRed  热度 ℃

分类

Android动画资源文件只有两种类型：

  • Property Animation
  • View Animation
    其中 View Animation 包括 Tween animation （补间动画）和 Frame animation（帧动画）。

  • Property Animation
    通过在一段时间内修改一个对象的属性值来创建一个动画。
  • View Animation
    —Tween animation：通过在单个图像上执行一系列转换来创建动画。
    — Frame animation：通过展示一系列有序的图片来创建动画。
